Instructions for wrestler program:

1) Make sure wrestler.py is in your folder.

2) Make sure you have one or more wrestler text files in your folder.
   Provided in my submitted files as wrestler.txt, wrestler1.txt, and wrestler2.txt to use.

3) Run the following command: python3 wrestler.py <name_of_file>
   Replace <name_of_file> with the name of your text file
   For example, if your file name is wrestler1.txt, run the following command:
       python3 wrestler.py wrestler1.txt

4) View the results of the program in the terminal.
